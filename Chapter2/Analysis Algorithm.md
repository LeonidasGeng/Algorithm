# 算法分析

算法分析重点关注**时间**与**空间**方面的资源

一个**问题**通常有**多种方法**可以解决，通过对这些方法**进行算法分析**，**选出一个最有效的算法**，或者**排除较差的算法**



评价一个算法

首先：

给出准确答案（功能是否正常）

其次：

优化解决问题的方法===>降低算法的复杂度（减少算法的运行时间）



## 算法的复杂度

### 时间复杂度

算法的执行时间（算法执行所需要的基本操作数 = 步骤）

### 空间复杂度

算法执行所需额外储存空间的大小



**分析算法的时间复杂度**都是考察**最坏情况**的**运行时间**
